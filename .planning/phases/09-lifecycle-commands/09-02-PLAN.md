    Create `apps/backend/src/lifecycle/setup.ts` exporting `async function runSetup(deps?: SetupDependencies): Promise<void>`.

    **Dependency injection interface (for testability, per auth-command.ts pattern):**
    ```typescript
    export interface SetupDependencies {
      runSubprocessFn?: typeof runSubprocess;
      checkDependenciesFn?: typeof checkDependencies;
      mkdirSyncFn?: typeof mkdirSync;
      unlinkSyncFn?: typeof unlinkSync;
      cpSyncFn?: typeof cpSync;
      writeFileSyncFn?: typeof writeFileSync;
      existsSyncFn?: typeof existsSync;
      resolveFn?: typeof resolve;
    }
    ```

    **Flow (mirrors omarchy setup.ts pattern):**

    1. `console.clear()`
    2. `p.intro('agent-bar setup')`
    3. Run dependency check via `checkDependenciesFn ?? checkDependencies`
       - If missing deps found: show `p.note()` listing each missing dep name + installHint
       - If any missing dep is 'Bun': `p.outro('Setup cannot continue without Bun')` and return
       - For non-critical missing deps (secret-tool, gnome-extensions): warn but continue
    4. `p.note()` explaining what setup will do:
       ```
       This will:
         1. Create ~/.local/bin/agent-bar CLI wrapper
         2. Install systemd user service + env override
         3. Copy GNOME extension + enable it
         4. Configure tmpfiles.d for runtime socket
       ```
    5. `p.confirm({ message: 'Apply setup now?', initialValue: true })`
       - If cancelled: `p.outro('Setup cancelled')` and return
    6. Spinner steps (use `const s = p.spinner()`):

       **Step: CLI wrapper**
       - `s.start('Creating CLI wrapper...')`
       - Get paths via `getInstallPaths()`
       - `mkdirSync(dirname(paths.cliSymlink), { recursive: true })`
       - Try `unlinkSync(paths.cliSymlink)` catch (remove existing)
       - Build wrapper content: `#!/usr/bin/env bash\nset -euo pipefail\nexec bun "${join(REPO_ROOT, 'apps', 'backend', 'src', 'cli.ts')}" "$@"\n`
       - Write wrapper with `Bun.writeSync(paths.cliSymlink, wrapperContent)` (note: NOT Bun.write -- this is a shell script, not JSON)
       - Make executable: `Bun.spawnSync(['chmod', '+x', paths.cliSymlink])`
       - `s.stop('CLI wrapper created')`

       **Step: Systemd service**
       - `s.start('Installing systemd service...')`
       - `mkdirSync(paths.systemdDir, { recursive: true })`
       - `cpSync(join(REPO_ROOT, 'packaging', 'systemd', 'user', 'agent-bar.service'), paths.serviceFile)`
       - `mkdirSync(paths.overrideDir, { recursive: true })`
       - Build env.conf content: `[Service]\n` + for each var in ENV_VARS_TO_CAPTURE, if `process.env[var]` exists, add `Environment=${var}=${process.env[var]}\n`
       - `writeFileSync(paths.envOverride, envContent)`
       - `s.stop('Systemd service installed')`

       **Step: tmpfiles.d**
       - `s.start('Configuring tmpfiles.d...')`
       - `mkdirSync(paths.tmpfilesDir, { recursive: true })`
       - `cpSync(join(REPO_ROOT, 'packaging', 'tmpfiles.d', 'agent-bar.conf'), paths.tmpfilesConf)`
       - Try: `await runSubprocess('systemd-tmpfiles', ['--user', '--create', paths.tmpfilesConf])` catch (best-effort)
       - `s.stop('tmpfiles.d configured')`

       **Step: systemctl reload + enable + restart**
       - `s.start('Enabling systemd service...')`
       - `await runSubprocess('systemctl', ['--user', 'daemon-reload'])`
       - `await runSubprocess('systemctl', ['--user', 'enable', 'agent-bar.service'])`
       - `await runSubprocess('systemctl', ['--user', 'restart', 'agent-bar.service'])`
       - `s.stop('Systemd service enabled and started')`
       - Wrap all three in try/catch; on failure: `p.log.warn('systemctl commands failed...')` but continue

       **Step: GNOME extension**
       - `s.start('Installing GNOME extension...')`
       - `mkdirSync(paths.extensionDir, { recursive: true })`
       - For each item in EXT_ITEMS: try `cpSync(join(REPO_ROOT, 'apps', 'gnome-extension', item), join(paths.extensionDir, item), { recursive: true })` catch (skip if missing)
       - Try: `await runSubprocess('gnome-extensions', ['enable', GNOME_EXT_UUID])` catch (warn user may need to enable manually)
       - `s.stop('GNOME extension installed')`

    7. Check if `~/.local/bin` is in PATH: `if (!process.env.PATH?.split(':').some(d => resolve(d) === resolve(dirname(paths.cliSymlink))))`
       - If not in PATH: `p.log.warn('~/.local/bin is not in your PATH. Add: export PATH="$HOME/.local/bin:$PATH"')`
    8. `p.outro('Setup complete')`
    9. Wrap entire flow after p.confirm in try/catch: on error, `p.outro('Setup failed: ' + message)` and `process.exitCode = 1` (NOT process.exit(1) -- agent-bar convention)

    **Anti-patterns to avoid:**
    - Do NOT use sudo anywhere
    - Do NOT use process.exit() -- use process.exitCode = 1 and return
    - Do NOT use ora -- use @clack/prompts spinner consistently
    - Do NOT create a symlink to cli.ts directly -- create a bash wrapper that invokes bun (matching current install-ubuntu.sh pattern)

    **Flow (mirrors omarchy update.ts pattern, adapted per CONTEXT.md locked decisions):**

    1. `console.clear()`
    2. `p.intro('agent-bar update')`
    3. Verify git repo: `runGit(['rev-parse', '--git-dir'])` -- if not ok: `p.log.error('Not a git repository')`, `p.outro('Update failed')`, return
    4. Get current info: `runGit(['branch', '--show-current'])` and `runGit(['rev-parse', '--short', 'HEAD'])`
       - `p.log.info('Branch: ...')` and `p.log.info('Current: ...')`
    5. **Check dirty state (LOCKED DECISION -- fail if local changes):**
       - `runGit(['status', '--porcelain'])`
       - If output is non-empty: `p.log.error('Working directory has uncommitted changes.')`, `p.log.warn('Commit or stash your changes first.')`, `p.outro('Update cancelled')`, return
    6. **Check for diverged history:**
       - `runGit(['rev-list', 'HEAD..origin/master', '--count'])` after fetch (do this after step 7)
    7. **Fetch:**
       - `const s = p.spinner()`
       - `s.start('Fetching latest changes...')`
       - `runGit(['fetch', 'origin'])` -- if not ok: `s.stop('Failed to fetch')`, show error, return
       - Check behind: `runGit(['rev-list', 'HEAD..origin/master', '--count'])`
       - If count is 0: `s.stop('Already up to date!')`, `p.outro('No updates available')`, return
       - `s.stop('Updates available')`
    8. **Show incoming commits (LOCKED DECISION):**
       - `runGit(['log', '--oneline', 'HEAD..origin/master', '-15'])`
       - `p.note(commitOutput, 'Incoming commits')` -- show commit summary
    9. **Confirm:**
       - `p.confirm({ message: 'Apply update?', initialValue: true })`
       - If cancelled: `p.outro('Update cancelled')`, return
    10. **Pull (LOCKED DECISION -- ff-only):**
        - `s.start('Pulling changes...')`
        - `runGit(['pull', '--ff-only'])` -- if not ok: `s.stop('Pull failed')`, show error, `p.log.warn('If you have local commits, try: git rebase origin/master')`, return
        - `s.stop('Code updated')`
    11. **Install dependencies:**
        - `s.start('Installing dependencies...')`
        - `await run('bun', ['install'], { cwd: REPO_ROOT, timeoutMs: 60_000 })` -- use runSubprocess
        - `s.stop('Dependencies updated')`
    12. **Restart service:**
        - `s.start('Restarting service...')`
        - Try: `await run('systemctl', ['--user', 'daemon-reload'])` then `await run('systemctl', ['--user', 'restart', 'agent-bar.service'])`
        - `s.stop('Service restarted')` -- wrap in try/catch, warn on failure

    14. **Show result:**
        - Get new commit: `runGit(['rev-parse', '--short', 'HEAD'])`
        - `p.log.success('Updated: oldCommit -> newCommit')`
    15. `p.outro('Update complete!')`
